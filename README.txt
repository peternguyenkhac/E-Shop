#E-Shop
Công nghệ sử dụng: ASP .NET, Entity Framework, SQL Server
Chức năng chính: 
-Người dùng:
	+Xem danh sách sản phẩm, tìm kiếm sản phẩm
	+Xem chi tiết sản phẩm
	+Thêm sản phẩm vào giỏ hàng, quản lý giỏ hàng
	+Thanh toán (VNPAY)
	+Quản lý đơn hàng
-Admin:
	+Quản lý sản phẩm
	+Quản lý danh mục
	+Quản lý Banner
	+Quản lý đơn hàng
	+Quản lý người dùng

*Phần Admin hiện tại chưa làm
